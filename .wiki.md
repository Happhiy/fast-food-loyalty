# Project Summary
The Fast-Food Loyalty System is an innovative platform designed for fast-food kiosks, enabling customers to earn loyalty points on purchases and redeem them for coupons. It includes an admin interface for managing customer accounts, tracking transactions, and overseeing the coupon system. Enhanced security features include secure login via loyalty ID and an 8-digit PIN, along with multi-language support for a better user experience.

# Project Module Description
- **User Roles**: 
  - **Admin/Staff**: Manage customer accounts, record purchases, and redeem coupons.
  - **Customer**: Check points, view purchase history, and redeem coupons after accumulating 100 points.

- **Point System**: 
  - Points awarded based on purchase amounts and user roles, with multipliers for different roles.

- **Coupon System**: 
  - Admins verify and redeem coupons, while customers can redeem coupons after accumulating sufficient points.

- **Pages**:
  - **/login**: User login page.
  - **/home**: Displays points, purchase history, and coupon redemption options.
  - **/coupons**: Lists available and redeemed coupons.
  - **/admin**: Admin dashboard for managing customers, purchases, and coupons.

- **routes/**: Defines API endpoint routes and their handlers.

# Technology Stack
- **Backend**: Node.js, Express, TypeScript, PostgreSQL, Prisma ORM, JWT for authentication
- **Frontend**: React, TypeScript, Axios for API calls, Tailwind CSS for styling
- **State Management**: Local Storage (for tokens)
- **Routing**: React Router
- **Linting**: ESLint

# Usage
1. Install dependencies:
   ```bash
   cd app/backend
   pnpm install
   ```

2. Configure environment variables:
   ```bash
   cp .env.example .env
   # Update .env with your PostgreSQL connection string and JWT secrets
   ```

3. Set up the database:
   ```bash
   pnpm run prisma:migrate
   pnpm run seed
   ```

4. Start the application:
   ```bash
   pnpm run dev
   ```
